 # Tenant Wise

  AI-powered legal document generation for small Texas landlords (1-10 units).

  ## What it does

  TenantWise helps landlords generate Texas Property Code-compliant documents in seconds — no lawyer required. Stop wrestling with templates or paying
   for bloated property management software.

  ## Features

  - **AI Document Generation** - Late rent notices, lease renewals, maintenance responses, move-in/out checklists, security deposit returns
  - **Texas-Compliant** - All documents follow Texas Property Code requirements
  - **Legal AI Assistant** - Chat with an AI trained on Texas landlord-tenant law (paid plans)
  - **Property & Tenant Management** - Save profiles for quick document generation
  - **Affordable Pricing** - Free tier available, paid plans starting at $19/month

  ## Tech Stack

  - **Framework**: Next.js 16 (App Router)
  - **Database**: Supabase (PostgreSQL)
  - **Auth**: Supabase Auth
  - **AI**: Claude API (Anthropic)
  - **Payments**: Stripe
  - **Styling**: Tailwind CSS

  | Basic | $19/mo | Unlimited docs, all templates, Legal AI Assistant |
